Lambda@Edge Image Pipeline

S3 → CloudFront → viewer-request rewrite → origin-response resize on 404.

Simulate a request

How it works

  1. Viewer request rewrites /images/… URIs using query params and Accept / User-Agent.
  2. CloudFront fetches the variant key from S3.
  3. Origin response on 404 loads the original, resizes with Sharp, caches, returns 200.
